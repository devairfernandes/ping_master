# 🚀 Ping Master - Release Note v1.0.0

A primeira versão estável do **Ping Master** traz uma interface de monitoramento de rede inspirada no painel de alta performance Ouromax-BL10, focada em simplicidade, design premium e atualização em tempo real.

## ✨ Destaques desta Versão

### 🎨 Interface e Experiência do Usuário (UX)
*   **Nome Oficial:** Implementação da marca **PING MASTER** na tela de entrada.
*   **Visual Pro:** Tema *Pure Black* (amigável a telas AMOLED) com acentos em verde neon.
*   **Layout Responsivo:** Estrutura baseada em grids inteligentes que se ajustam a qualquer tamanho de tela de smartphone (vertical ou horizontal).
*   **Dashboard Compacto:** Cards de resumo (Online, Ping Médio, Último Update) com ajuste automático de texto (*FittedBox*).

### ⚙️ Funcionalidades Técnicas
*   **Conexão por Endpoint:** Tela inicial que permite ao usuário digitar o IP/URL e a porta do servidor de monitoramento.
*   **Gráficos em Tempo Real:** Visualização de histórico de ping através de curvas de Bezier suaves e gradientes dinâmicos.
*   **Detecção de Jitter:** Monitoramento de instabilidade de rede com indicadores visuais (Excelente / Bom / Alerta).
*   **Sistema OTA (Over-The-Air):** Verificação automática de novas versões via GitHub integrada ao Dashboard.

## 📦 Detalhes de Instalação
*   **Versão:** 1.0.0
*   **Tamanho do APK:** ~16.3 MB
*   **Arquitetura:** ARM64-v8a
*   **GitHub Oficial:** [devairfernandes/ping_master](https://github.com/devairfernandes/ping_master)

## 🛠️ Como Atualizar Futuramente
1.  Subir o novo APK nas *Releases* do GitHub.
2.  Atualizar o arquivo `version.json` no repositório com a nova versão e o link de download direto.
3.  O app detectará a mudança automaticamente e oferecerá a atualização para o usuário.
