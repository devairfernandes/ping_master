# 🚀 Ping Master - Release Note v1.0.3

Esta versão resolve um dos problemas mais solicitados: a persistência das configurações de conexão.

## ✨ Novidades da Versão 1.0.3

### 💾 Persistência de Dados
*   **Memória de Conexão:** O aplicativo agora utiliza `shared_preferences` para salvar a última URL de servidor utilizada.
*   **Auto-Preenchimento:** Ao abrir o app ou voltar para a tela de login, o servidor que você configurou anteriormente já estará preenchido automaticamente.
*   **Configuração Inteligente:** Alterações feitas dentro do Dashboard também são salvas permanentemente.

### 🛡️ Estabilidade
*   Ajuste na lógica de transição entre telas para garantir que a URL correta seja sempre carregada.
