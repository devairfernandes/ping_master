# 🚀 Ping Master - Release Note v1.0.1 (Atualização Crítica)

Esta atualização foca na automação de distribuição e na flexibilidade do administrador, introduzindo o sistema de atualização Over-The-Air (OTA) e melhorias na configuração de rede.

## ✨ Novidades da Versão 1.0.1

### 🔄 Sistema de Atualização OTA (Over-The-Air)
*   **Verificação Automática:** O app agora consulta o GitHub ao iniciar para verificar se há uma nova versão disponível.
*   **Gerenciador de Download Interno:** Interface nativa para download do APK com barra de progresso em tempo real.
*   **Instalação Facilitada:** Integração com `open_filex` para disparar a instalação do pacote Android imediatamente após o download.

### ⚙️ Flexibilidade e Conectividade
*   **Servidor Editável:** Adicionada a capacidade de alterar o IP/URL do servidor diretamente do Dashboard, sem precisar deslogar.
*   **Persistência de Conexão:** Melhoria na estabilidade do timer de monitoramento.
*   **Fallback Inteligente:** Otimização dos dados de exemplo quando o servidor está offline, permitindo testar a interface mesmo sem conexão ativa.

### 🛡️ Engenharia e Performance
*   **Novas Dependências:** Inclusão de `dio` para downloads robustos e `package_info_plus` para gestão de versionamento.
*   **Otimização de Build:** Redução no tempo de resposta da interface durante a atualização de dados.
