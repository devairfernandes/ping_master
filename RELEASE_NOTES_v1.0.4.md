# 🚀 Ping Master - Release Note v1.0.4

Esta versão foca na transparência do estado de conexão e na robustez da comunicação com o servidor.

## ✨ Novidades da Versão 1.0.4

### 🔍 Diagnóstico de Conexão
*   **Banner de Modo Offline:** Agora o app avisa explicitamente quando não consegue conectar ao servidor e está exibindo dados de demonstração.
*   **Aumento de Timeout:** Tempo de espera para resposta do servidor aumentado de 3 para 7 segundos, melhorando a conexão em redes instáveis.

### 📊 Exibição de Dados
*   **Carregamento Dinâmico:** Removida qualquer limitação de quantidade de hosts. O Dashboard agora renderiza todos os serviços que o servidor enviar via API.
*   **Rodapé Informativo:** Agora é possível visualizar a versão do app diretamente no final do Dashboard.
